# To Do App

## stack
```sh
-NodeJS
-Express
-JWT
-SQLite3
-Javascript
-HTML5/CSS
-Jest
-Supertest
-TypeScript
```

## Launch backend server
```sh
    cd backend
    npm start
```

## Launch backend tests
```sh
    cd backend
    npm test
```

## Launch frontend test
```sh
    cd frontend
    npm test
```
